# Script-Python-auto-reorganize
Script en Python qui réorganise un dossier selon la date de création des fichiers : si un fichier est vieux de plus de 2 semaines, le déplace dans un sous-dossier du nom du créateur.
Script in Python that reorganizes a folder depending on creation time


ATTENTION : SI VOUS UTILISEZ LE PROGRAMME --> faites un test pour voir si le dossier se créer bien tout seul, sinon créez un dossier du nom de chaque utilisateur qui aura des fichiers dans le dossier à trier,
autrement ces fichiers seront détruits !
